# Text / Content Rules

## Do Not Write

- купи
- продай
- лонг
- шорт
- точка входа
- прибыль гарантирована
- гарантированный доход
- инвестиционный совет
- финансовая рекомендация

## Preferred Wording

- выбор
- сценарий
- риск
- последствие
- хаос
- сезон
- игровой результат
- событие
- окно
- бонус
- решение

## Fictionalization

- All tickers are fictional.
- All exchanges are fictional.
- All influencers are fictional.
- All rooms, players, rankings, rewards, losses, and market events are game data.
- Copy must reinforce that Crypto Reality is a game, not a trading terminal.

## CTA Style

- Use: `Choose`, `Continue`, `Join room`, `Start game`, `Share result`, `Review risk`, `Submit choice`.
- Avoid action labels that imply real trading or financial instructions.
